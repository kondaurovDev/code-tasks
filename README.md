## Как работать с задачами

```bash
# Установить зависимости (один раз)
npm install

# Запустить все тесты
npm test

# Запустить конкретную задачу (например, первую)
npm test 01

# Режим watch - тесты перезапускаются при изменении кода
npm run test:watch
```

**Файлы задач находятся в папке `tasks/`** - открой нужный файл и напиши реализацию функции.
